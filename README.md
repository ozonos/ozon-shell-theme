Ozon for Gnome Shell
====================

### Compatibility

Gnome 3.10 and 3.12

### Installation

Place the `Ozon` directory in `/usr/share/themes/` or `~/.local/share/themes/`

The theme requires [User Theme Extension](https://extensions.gnome.org/extension/19/user-themes/) for Gnome Shell.
